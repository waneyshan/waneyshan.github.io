Mon, Jan 1, 0001 Django基础 文档链接 新建项目 略 新建app 略 Channels + Django 实现Websock 文档链接 安装channels pip install channels 配置Django的settings.py文件 在INSTALL_APPS字段添加channels 如下, 尽量添加在其他app的前面： INSTALLED_APPS = ( 'channels', 'django.contrib.auth', 'django.contrib.contenttypes', 'django.contrib.sessions', 'django.contrib.sites', ··· ) 在项目同名目录下新建文件routiong.py 也即，routing.py和settings.py在同一目录下。 【注1】：规定目录只是方便管理，本质上无论放在哪个位置都可以 【注2】：routing相关模块的功能和urls相关模块的功能相似， from channels.routing import ProtocolTypeRouter application = ProtocolTypeRouter({ # Empty for now (http->django views is added by default) }) 配置settings.py文件 添加配置项如下： ASGI_APPLICATION = "myproject.routing.application" 其中，‘myproject’为你的项目名称，整个路径就是上一步文件中定义的application的路径。
